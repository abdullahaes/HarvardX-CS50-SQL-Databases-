# 🏛️ Harvard CS50 SQL - My Journey into Databases

Welcome to my personal repository for the **CS50's Introduction to Databases with SQL** course from Harvard University. This repository documents my progress and projects as a 3rd-year Computer Engineering student.

## 📁 Projects Structure

The repository is organized into distinct projects:

### 1. 🔍 Cyberchase Project (`cyberchase_project/`)
Data retrieval and filtering using the `SELECT`, `WHERE`, and `LIMIT` clauses.

### 2. 🗻 36 Views Project (`views_project/`)
Connecting Art with Computational Image Analysis using statistics like brightness and entropy.

### 3. ⚾ Players Project (`players_project/`)
Analyzing MLB player data from 1871 to 2023 with complex sorting and filtering.
